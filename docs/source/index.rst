TotemBionet Documentation
==========================


TotemBionet relies on Docker_ and Jupyter_ technologies to provide an
unified Python_ environment to to edit, execute, share, and reproduce analyses
of qualitative models of biological networks.

.. _Docker: https://www.docker.com/
.. _Jupyter: https://jupyter.org/
.. _Python: https://www.python.org/
